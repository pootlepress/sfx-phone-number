#SFX Telephone
================================

A simple plugin boilerplate to fast-track your Storefront extension development.

To get started: change the directory name, sfx-telephone.php, languages/sfx-telephone.po to reflect your new plugin name.

Then do some Find/Replace (case sensitive) across the entire directory:

* 'sfx-telephone' => 'your-extension'
* 'SFX_Telephone' => 'Your_Extension'
* 'SFX Telephone' => 'Your Extension'
* 'sfx_telephone_' => 'your_extension_'
* 'sfxtp_' => 'ye_'
* 'sfxtp-' => 'ye-'

That's it! Start building!
